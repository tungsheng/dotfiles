# Dotfiles

Tony's dotfiles for Alacritty, Bash, Tmux and Neovim. The minimalism config style is influenced by Mischa's dotfiles.


# Setup

## macOS

```bash
$ git clone https://github.com/tungsheng/dotfiles.git $HOME/
$ cd $HOME/dotfiles
$ ./setup
```
