# Dotfiles [WIP]

Tony's dotfiles. Including config setup and server setup.

## Getting Started

### Installing

Step 1: clone this repo to your computer or server.

```
$ git clone https://github.com/tungsheng/dotfiles.git
```

Step 2: run installation script

```
$ cd dotfiles
$ ./install.sh
```
